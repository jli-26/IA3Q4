## AI Disclosure
I used AI (ChatGPT) to assist with debugging and resolving a Material 3 compilation issue. Specifically, I encountered an experimental API error when using TopAppBar and NavigationRail. The AI suggested adding @OptIn(ExperimentalMaterial3Api::class) to acknowledge the use of experimental Material 3 components. I reviewed and implemented the suggestion myself, ensured it was appropriate for my project, and verified that the layout and functionality still met the assignment requirements.
